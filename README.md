# L'Alarme domestique

## Introduction
Ce projet conciste à reproduire une alarme domestique à l’aide d’une carte Arduino et quelques autres composants. J'ai utilisé les composants suivant:
* Détecteur de mouvement
* Ecran LCD I2C
* Clavier numérique
* Buzzer
* LED rouge
